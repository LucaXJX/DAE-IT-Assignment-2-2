## 使用說明

### 環境要求

- Node.js 16.x 或以上
- npm 或 pnpm

### 安裝與運行

```bash
# 安裝依賴
npm install

# 編譯 TypeScript 代碼
npm run build

# 啟動服務器
npm start
```

### 訪問應用

瀏覽器打開: http://localhost:8080
